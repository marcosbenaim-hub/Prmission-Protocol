What Prmission Does

Prmission is a consent-gated escrow and settlement protocol for AI agent commerce. It gives agents a standard flow to request permission, lock funds, and settle outcomes on-chain.

Neutral Infrastructure

Prmission integrates with model and interface layers without replacing them.

Immutable Monetization

3% protocol fee (300 bps) is hardcoded on-chain and captured on settlement.

Compounding Volume

As settlement-required agent actions scale, protocol fee capture scales with them.

Proof

Live Deployment

Verified contract on Base mainnet. Settlement token is USDC on Base.

How It Works

1. grantPermission()User defines scope and fee
2. depositEscrow()Agent locks USDC
3. reportOutcome()Outcome is reported
4. disputeSettlement()24-hour dispute window
5. settle()Funds auto-distribute
