What Prmission Does

Prmission is a consent-gated escrow and settlement protocol for AI agent commerce. It gives agents a standard on-chain flow to request permission, lock funds, and settle outcomes — deployed on Base (Ethereum L2) with Ethereum ERC-8004 identity verification.

Neutral Infrastructure

Prmission integrates with model and interface layers without replacing them. We settle on-chain beneath them.

Immutable Monetization

3% protocol fee is hardcoded on-chain in the Prmission smart contract on Base (Ethereum L2). Not even we can change it.

Compounding Volume

As settlement-required agent actions scale, on-chain protocol fee capture scales with them.

How It Works

1. grantPermission()User defines scope and fee — Ethereum ERC-8004 verified
2. depositEscrow()Agent locks USDC on-chain
3. reportOutcome()Outcome reported on-chain
4. disputeSettlement()24-hour on-chain dispute window
5. settle()Funds auto-distribute on-chain
